Capreze v{{BUILD_VERSION}}

キャプチャに最適化されたウィンドウ サイズ変更ツール

スクリーンショットを取るときにウィンドウサイズと画像サイズに差異があることに困っていませんか？
Capreze は自動的にオフセットを検出し正確なサイズでキャプチャができるようにします。
マルチモニターにも対応しており、対象のウィンドウが表示されているモニターに合わせて、オフセットを調整することができます。

【動作環境】
本ソフトウェアは Windows 10 または Windows 11 で動作します。
また .NET 8 Runtime (https://dotnet.microsoft.com/ja-jp/download/dotnet/8.0) をインストールする必要があります。

【インストール方法】
capreze_{{BUILD_VERSION}}.msi を実行してください。

【アンインストール方法】
スタートメニューのCaprezeを右クリックしアンインストールをクリックしてください。
