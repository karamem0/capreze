Capreze v{{BUILD_VERSION}}

キャプチャ精度を高めるために、ウィンドウの表示サイズと実際の画像サイズとの差異 (オフセット) を自動検出し、指定どおりのサイズでキャプチャできるようにするツールです。
マルチ モニター環境では、対象ウィンドウが表示されているモニターごとに補正を適用できます。

【主な機能】
- ウィンドウと取得画像の境界差分 (オフセット) 自動検出
- マルチ モニター対応
- 指定ピクセル単位の正確なサイズ設定支援

【インストール手順】
1. .NET 8 Runtime がインストール済みであることを確認します。
2. capreze_{{BUILD_VERSION}}.msi をダウンロードします。
3. ファイルをダブルクリックします。
4. 画面の指示に従って完了します。

【アンインストール手順】
1. スタート メニューを開きます。
2. Capreze を右クリックします。
3. アンインストール を選択します。
4. 表示される指示に従います。
